# Álbum Mundial 2026 — Ala personal

Sistema tipo Profiler (medir / cuantificar / seguir) para **completar el álbum Panini FIFA World Cup 2026** — el último Mundial de Cristiano Ronaldo, héroe personal de Boris — de forma óptima en costo.

## Objetivo
Cerrar las 980 láminas gastando lo mínimo, usando el modelo del **coleccionista de cupones**: sobres hasta ~80-85% → canje de repetidas + servicio oficial de faltantes.

## Archivos
| Archivo | Qué es |
|---------|--------|
| `DASHBOARD.md` | KPIs, ficha verificada, modelo de decisión, registro de compras |
| `CHECKLIST.md` | Inventario equipo por equipo (✅/❌/🔁) |
| `fotos/` | Fotos fuente del coleccionista |
| `crops/` | Recortes de trabajo para leer láminas |

## Workflow
1. **Inventario:** Boris fotografía de frente cada página/equipo → Claude lee láminas y marca estado.
2. **Repetidas:** foto en abanico o dictado → moneda de canje.
3. **Precios/valor:** búsqueda web (DR) → enciende el modelo de costo.
4. **Decisión:** Claude calcula punto de cruce y costo para cerrar.
